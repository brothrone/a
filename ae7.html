01 Lofting의 정의

Lofting: 항공기의 외부 형상(external geometry)을 정의하는 과정.
좋은 레이아웃의 3조건: ① 매끄러운 공력 형상 제공 ② 내부 구성품(payload, engine, landing gear 등) 명확히 배치 ③ 이후 상세설계에서 큰 변경 불필요.
항공기 개념설계의 핵심(heart and soul)이며, 공력·구조 경험이 바탕이 되어야 함.

02 Configuration Layout
산출물 (End Products)
  1. 1
    Conceptual Sketch — 전체 공력 배치(fuselage, wing, tail) + 주요 내부 구성품 위치 표시. 설계 시작점
  2. 2
    Design Layout (CAD) — 분석에 필요한 기하 정보 포함. 날개·꼬리 사다리꼴 geometry, 면적·부피 측정값 포함
  3. 3
    Wetted Area Plot — 동체 station별 단면 둘레를 플롯 → 마찰 항력 추정에 사용
  4. 4
    Volume Distribution Plot — 단면적 vs 종방향 위치 → 초음속 wave drag 예측에 사용
  5. 5
    Inboard Profile Drawing — 수차례 반복 최적화 후 작성; 내부 서브시스템 배치 상세 도면. Preliminary design에서 완성
  6. 6
    Inboard Isometric Drawing — art group이 브리핑·제안용으로 작성하는 3D 투시도

※ 설계자가 직접 준비해야 하는 3가지 입력값: wetted area / volume distribution plot / fuel-volume plots

03 7.8.1 Reference Wing/Tail
MAC 관련 수식

MAC(Mean Aerodynamic Chord, C̄): 날개 전체 면적이 그 한 점에 압축된 것처럼 작용하는 등가 시위.
Ȳ: MAC의 스팬 방향 위치. MAC의 25% 지점 = 날개 전체의 공력 중심(AC).

사다리꼴 날개 (Trapezoidal Wing) b = √(A·S)    (7.5) 스팬
C_root = 2S / [b(1+λ)]    (7.6) 루트 코드
C_tip = λ · C_root    (7.7) 팁 코드
C̄ = (2/3) · C_root · (1+λ+λ²)/(1+λ)    (7.8) MAC
Ȳ = (b/6) · (1+2λ)/(1+λ)    (7.9) MAC 위치
타원형 날개 (Elliptical Wing) C_root = 4S / (π·b)    (7.10a)
C(y) = C_root · √[1-(y/(b/2))²]    (7.10b)
타원형 날개: MAC = 84.9% of C_root, Ȳ = 52.9% of semispan
04 7.8.2 Wing Location
CG–MAC 위치 관계

날개 위치(종방향)는 MAC 대비 CG 위치(%)로 결정된다. 이것이 종방향 안정성의 핵심.

항공기 형태 CG 위치 (MAC 기준) 비고
순수 비행익 (Pure Flying Wing) 25% MAC 중립 안정 (neutral stable)
안정형 + 후방 꼬리 (Stable, aft tail) 30% MAC B-747, C-5 등 일반 수송기
불안정형 + 후방 꼬리 (Unstable, aft tail) 40% MAC F-22 등 FBW 전투기
카나드형 (Control-type canard) 15~20% MAC FBW 불안정 항공기

※ Static Margin = X̄_np − X̄_cg  |  C_mα = −C_Lα(X̄_np − X̄_cg) → 음수일수록 안정

Pitching Moment Derivative (C_mα): 받음각 변화에 대한 피칭 모멘트 변화율. C_mα < 0 → 종방향 안정. 수송기(−1.0 수준), 전투기(−0.1~0 근처). Static margin이 클수록 안정하나 조종성 저하.

05 7.8.3 Reference Wing vs
Exposed Wing 차이

Reference (Trapezoidal) Wing

  • 항공기 중심선(centerline)까지 연장한 가상의 날개
  • 상반각(dihedral)이 평면도(top-view)에 영향 없음
  • 공력 계수 계산의 기준면적(S_ref)
  • AR, λ, sweep 등 기하 파라미터 정의 기준

Actual Exposed Wing

  • 동체 측면(side of fuselage)에서 시작
  • 상반각이 실제 날개 면적을 증가시킴
  • 실제 구조 설계·공력 해석의 대상
  • 사다리꼴이 아닐 수 있음 (nontrapezoidal)
06 7.8.3 Nontrapezoidal Wing
변형 4가지 (Fig. 7.25)

a) Rounded Wingtip

  • 날개 팁을 둥글게 처리
  • 팁 와류 감소 → 유도 항력 경감

b) Trailing-Edge "Kick/Bat"

  • 뒷전을 바깥쪽으로 꺾음
  • 플랩 코드 증가 + 랜딩기어 수납 두께 확보

c) Strake / LEX

  • Leading-Edge Extension
  • 전투기 고기동 시 양력 증가
  • 예: F-16, F/A-18

d) Curved (Delta)

  • 앞전이 곡선 형태
  • 천음속·초음속 충격파 최소화
07 7.8.3 Wing Rigging

Wing Rigging: 에어포일 단면을 수직 방향으로 이동시켜 원하는 스팬 방향 직선(hinge line 등)을 만드는 과정.
복잡한 날개 곡률 → aileron·flap의 힌지 라인이 곡선이 될 수 있음 → 곡선 힌지 라인은 구현 불가.
해결: 중간 스팬 에어포일을 수직으로 조금 내리면 힌지 라인이 직선(Section B-B)이 됨 → aileron·flap 설계 가능.

08 7.8.4–5 에어포일 보간법
Linear vs Flat-Wrap

Linear Interpolation Ruled Surface

  • 루트·팁 에어포일의 "가중 평균"으로 중간 단면 생성
  • 같은 코드 퍼센트(%) 점끼리 스팬 방향 연결
  • 비틀림·캠버 차이 시 복합 곡률 발생 가능
  • 제작 시 평판으로 만들 수 없는 면 생길 수 있음

Flat-Wrap Interpolation Same Slope

  • 같은 코드 %가 아닌 같은 표면 기울기(tangent angle)를 가진 점끼리 연결
  • 평판을 감아서(flat-wrap) 만들 수 있는 곡면 보장
  • 비틀린 날개나 형상이 다른 루트·팁에 필수
09 7.8.7 Wing Fillet

Wing Fillet(루트 페어링): 날개-동체 교차부를 원호(circular arc)로 매끄럽게 블렌딩하는 처리. 공력 간섭 항력 감소 목적.
반경: 루트 코드 길이의 약 10%. 항공기 후방으로 갈수록 반경이 커져 기류 박리 최소화.
초기 레이아웃에서는 경험적으로("eyeballed") 처리하는 경우 많음.

10 7.8.8 Winglet Design
  1. 목적
    유도 항력 감소 — 날개 팁 와류 약화 → 항력 감소량은 윙렛 높이에 비례
  2. 위치
    팁 에어포일 최대 두께 지점 — 거기서 시작; 날개와 유사한 후퇴각 적용
  3. 높이
    팁 코드(c_t) 이상 — 높을수록 항력 감소 효과↑; Height = c_t 가 기준
  4. 캠버
    날개보다 큰 캠버 — 충분한 사이드 포스 확보; 앞전 외향 4° 받음각 설정
  5. 두께
    t/c ≈ 8% — 대표 설계값
  6. 하부 패널
    항력 감소 기여 적음 — 지상 접지 위험으로 생략 경우 많음. 포함 시 루트 7°, 팁 11° 비틀림
  7. 주의
    Flutter (플러터) — 윙렛은 탄성축 뒤에 질량 추가 → 반드시 aeroelastic 해석 수행
11 7.9 Wetted Area
(젖은 면적) 결정

S_wet: 항공기 외부 총 노출 표면적. 마찰 항력(friction drag)의 주요 원인으로 항력 추정의 핵심 입력값.

  1. 날개·꼬리
    두께비 기반 공식 — t/c < 0.05: S_wet = 2.003·S_exposed  |  t/c > 0.05: S_wet = S_exposed[1.977 + 0.52(t/c)]
  2. 동체
    측면·상면 투영 면적 평균 활용 — S_wet ≈ 3.4·(A_top + A_side)/2  (Eq. 7.13)
  3. 정밀법
    단면 둘레의 종방향 적분 — 각 station의 단면 둘레를 플롯 → 곡선 아래 면적 = 젖은 면적

※ S_exposed = 투영 면적(top-view) ÷ cos(dihedral angle)

12 7.10 Volume Determination
(부피 결정)

내부 부피는 설계의 합리성 검증 지표. 통계적 기준보다 내부 부피가 작으면 개발 문제 및 정비성 저하 발생.

  1. 추정식
    Vol ≈ 3.4 · (A_top)(A_side) / 4L — 단면이 정사각형과 원의 중간 형태 가정. L: 동체 길이 (Eq. 7.14)
  2. 정밀법
    단면적의 종방향 적분 — 단면적 vs 종방향 위치 플롯 → 곡선 아래 면적 = 부피
  3. Volume Distribution Plot 활용
    초음속 wave drag 예측 — 부피 분포 형상이 초음속 항력 결정 (Area Rule과 연계, Ch.12)
13 7.8.6 Wing Cross-Section
내부 구성품 배치 확인

설계 중 중요한 과제: 연료탱크, 랜딩기어, 기타 내부 구성품이 날개 윤곽선 안에 모두 들어가는지 확인.
방법: 에어포일을 날개 평면도 위에 그린 후, 윙 기준면(Wing Reference Plane)에 수직한 단면도(cross-section)로 전개. 상반각 영향을 포함하여 실제 공간 확인.

01 에어포일 기하학
기본 정의
  1. 1Mean Camber Line — 앞전(LE)과 뒷전(TE)을 연결하는 chord line과 에어포일 상·하면 사이 중간선. Camber = 최대 이격 거리(chord 대비 %)
  2. 2Chord line — LE ~ TE 직선; chord 길이 c 는 기준 치수
  3. 3두께비 t/c — 최대 두께 / chord 길이. 에어포일 성능·구조를 지배하는 핵심 파라미터
  4. 4Leading Edge Radius — LE 곡률 반경. 두꺼울수록 크고, 실속 특성에 결정적 영향
  5. 5Symmetrical airfoil — camber = 0, 대칭. zero AoA에서 양력 = 0; 꼬리날개에 사용
  6. 6Cambered airfoil — camber > 0; zero AoA에서도 양력 발생. 설계 양력 계수 결정
02 에어포일 공력 계수
(section 계수)

에어포일(2D) 계수는 단위 스팬 기준이며 chord c를 기준 치수로 사용한다.

Cl = Section Lift / (q · c)
Cd = Section Drag / (q · c)
Cm = Section Pitching Moment / (q · c²) 단위 스팬 기준

소문자 l, d, m → 2D section 계수 (대문자 L, D, M은 3D 날개 계수)

03 Aerodynamic Center vs
Center of Pressure

Aerodynamic Center (AC) ★출제

  • 피칭 모멘트가 AoA에 무관하게 일정한 점
  • 아음속: chord 앞에서 25% 위치
  • 초음속: chord 앞에서 35~40% 위치
  • 설계 기준점으로 사용 (CG 위치 결정에 활용)

Center of Pressure (CP)

  • 수직력(양력)의 합력 작용점
  • AoA 변화에 따라 이동 → 기준점으로 부적절
  • 캠버드 에어포일: AoA↑ → CP 앞으로 이동
  • 설계에서 잘 사용하지 않음
04 Design Lift Coefficient
Cl,design

Cl,design은 L/D가 최대가 되는 양력 계수. Drag Polar 그래프에서 원점에서 그은 접선의 접점이 바로 Cl,design 이다. 에어포일의 Camber(캠버)에 의해 결정된다.

Cl,design → tangent point from origin on the Cd–Cl² polar
Camber ↑ → Cl,design ↑ (더 높은 순항 양력 계수)

설계 시: 순항 CL이 Cl,design과 일치하도록 에어포일 선정 → 최소 항력

05 실속 유형 3가지
(두께비에 따라)

① Fat Airfoil
t/c > ~14%

  • 뒷전에서부터 실속 진행
  • 완만하고 점진적 (gradual stall)
  • LE radius 커서 분리 늦음
  • 저속기·GA에 유리

② Medium Airfoil
t/c 6~14%

  • 앞전 기포(LE separation bubble)에서 실속
  • 급격하고 불연속적 (abrupt stall)
  • 조종사 경고 없이 실속 → 위험
  • 주의 필요

③ Very Thin
t/c < 6%

  • 앞전 코 분리 (nose separation)
  • Cl은 부드럽게 감소
  • 그러나 Cm 급변 → 조종성 문제
  • 초음속 날개에 사용
06 두께비 t/c 의
3가지 트레이드오프
  1. 1항력 — t/c ↑ → 파형(wave) 항력 및 마찰 항력 증가; 고속기일수록 t/c를 최소화
  2. 2임계 마하수 Mcr — t/c ↑ → Mcr ↓ (날개 위 흐름이 더 빨리 음속 돌파). 고속 설계에서 t/c 감소 필요
  3. 3구조 중량 — 구조 중량 ∝ 1/√(t/c). t/c ↑ → 구조적으로 유리 (날개보 높이 증가). 저속·대형기는 t/c를 어느 정도 크게

★ 세 가지가 서로 반대 방향 → 최적 t/c는 임무·속도에 따라 결정

07 에어포일 계열
종류 및 특징
계열 특징 주 용도
NACA 4-digit
(ex: 2412)		 앞 두 자리 → 캠버·위치, 뒤 두 자리 → t/c. 단순, 검증 풍부 꼬리날개, 일반 저속기
NACA 5-digit 최대 캠버 위치를 앞으로 이동 → Cl,max 향상. 그러나 실속 특성 나쁨 일부 GA·구형 수송기
NACA 6-series
(ex: 64A-series)		 층류 버킷(laminar bucket) 설계. 저항력 감소 범위 존재. 64A → 고속 날개 아음속 수송기, 전투기 날개
Supercritical Airfoil ★ 위면을 납작하게 설계 → 충격파 강도 감소 → Mcr 상승. 동일 속도에서 t/c 증가 가능 현대 수송기(B737, A320 등)
Laminar Flow Airfoil 최대 두께를 뒤로 이동 → 층류 영역 확장 → 마찰 항력 감소 소형 저속기, 일부 비즈니스 제트
Biconvex / Diamond 매우 얇은 초음속 전용. 아음속에서 성능 열악 초음속 전투기·미사일
08 Reference (Trap) Wing
기준 날개 정의

실제 항공기 날개는 동체를 통과하므로 형상 정의가 복잡하다. Reference Wing(기준 날개)은 동체를 무시하고 사다리꼴(trapezoidal) 날개를 항공기 centerline까지 연장한 가상의 날개다. 모든 날개 면적·기하 파라미터는 이를 기준으로 정의한다.

Wing Area S = Reference (trap) wing area (fuselage portion 포함)
b = tip-to-tip span (centerline 기준)
λ = C_tip / C_root (taper ratio)
AR = b² / S (aspect ratio)
09 MAC·날개 AC
위치 공식
MAC (Mean Aerodynamic Chord):
c̄ = (2/3) · C_root · (1 + λ + λ²) / (1 + λ)

MAC 스팬 방향 위치:
Ȳ = (b/6) · (1 + 2λ) / (1 + λ)

날개 AC 위치 (아음속): 25% MAC
날개 AC 위치 (초음속): 30~40% MAC Raymer §4.3

CG는 날개 AC 근처에 위치시켜야 안정성·조종성 확보 가능

10 Aspect Ratio (AR)
효과와 항공기별 값

AR은 유도 항력 및 실속 특성에 직접적 영향을 준다.

CD,i = CL² / (π · e · AR) → AR ↑ : 유도항력 감소
실속 각도: AR ↓ → 실속 각 증가 (날개 끝 효과) e = Oswald efficiency
항공기 유형AR 범위
활공기(Glider)20 ~ 30
GA 단발6 ~ 8
GA 쌍발7 ~ 9
수송기(Transport)7 ~ 12
군용 훈련기5 ~ 8
전투기(Fighter)2 ~ 4
초음속 전투기1.5 ~ 3
11 Wing Sweep (후퇴각)
Λ_LE vs Λ_C/4
  1. 1아음속 기준: Λ_C/4 — 1/4 chord 선 기준 후퇴각. 피칭 모멘트 및 공력 중심 분석에 사용
  2. 2초음속 기준: Λ_LE — LE가 Mach cone 안쪽에 위치해야 파형 항력 감소 → 앞전 후퇴각으로 정의
  3. 3Mcr 증가 효과 — 법선 마하수 M_n = M · cosΛ → 후퇴각 증가 시 유효 마하수 감소 → Mcr ↑
  4. 4Variable-sweep wing — 저속↔초음속 전환. 중량 페널티: 공허 중량 +4%, 날개 중량 +19% 페널티 암기
M_n = M · cos(Λ_LE) → Mcr,swept = Mcr,unswept / cos(Λ_LE)
12 Taper Ratio λ
테이퍼비 선정

λ = C_tip / C_root. 타원형 양력 분포(최소 유도 항력)를 근사하는 최적 λ가 존재하며, 후퇴각과 연계하여 선정한다.

λ ≈ 0.45 → 직선 날개에서 타원 양력 분포 근사 (최적)
저후퇴각(≤20°): λ = 0.4 ~ 0.5
고후퇴각(>25°): λ = 0.2 ~ 0.3 ★ λ < 0.2 피할 것 (팁 실속 위험)

δ (spanwise efficiency 보정) < 0.2 되면 팁 실속 발생 가능 — 피해야 함

13 Wing Twist
비틀림 종류

Geometric Twist (기하 비틀림)

  • chord line의 실제 받음각 변화
  • 루트→팁 방향으로 받음각 감소
  • Washout(세척 비틀림): 팁 LE 아래 방향 → 팁 받음각 감소
  • 일반적으로 약 −3° (washout 방향)

Aerodynamic Twist (공력 비틀림)

  • Zero-lift angle 차이로 정의
  • 에어포일 계열을 루트/팁 다르게 사용
  • 기하 비틀림과 결합하여 설계
  • 스팬 방향 양력 분포 최적화

Washout(−3°)의 목적: 루트가 먼저 실속 → 팁 실속 방지 → 에일러론 효율 유지

14 Wing Incidence
날개 장착각

동체 기준선(fuselage reference line) 대비 날개 chord line의 각도. 순항 시 동체를 수평으로 유지하면서 필요한 양력을 얻기 위해 설정. 순항 항력 최소화가 목적.

항공기 유형전형적 장착각
GA (일반 항공기)약 +2°
수송기 (Transport)약 +1°
군용기 (Military)약 0°
15 Dihedral (상반각)
안정성 효과
  1. 1Dihedral ↑ — 횡적 정안정성(lateral static stability) 증가. 뱅크 시 복원 모멘트 발생
  2. 2후퇴각과 등가 — 후퇴각 10° ≈ 상반각 1°의 안정성 효과 (고후퇴각 날개는 상반각 줄임)
  3. 3전진익(Forward Sweep) — 후퇴각 효과가 반대 → 횡안정성 감소 → 음의 상반각(anhedral) 적용
  4. 4저익 vs 고익 — 저익기: 상반각 필요(+5~7°). 고익기: 상반각 불필요 또는 약간의 anhedral
16 꼬리날개 기능
3가지
  1. 1Trim (트림) — 원하는 비행 조건에서 모멘트 합력 = 0 되도록 힘 제공. 속도·CG 변화에 대응
  2. 2Stability (안정성) — 외란 발생 시 원래 평형 상태로 복원하는 모멘트 제공 (pitch, yaw, roll)
  3. 3Control (조종) — 조종사 입력에 따라 원하는 자세·운동 유발 (elevator, rudder, aileron)

18 Canard 두 종류
Control vs Lifting

Control Canard 조종 전용

  • 피치 조종 역할만 담당 (trim 불포함)
  • 낮은 AR, 작은 면적, 캠버 없음(대칭)
  • 순항 시 거의 양력 생산 안 함
  • 일부 전투기·미사일에 사용

Lifting Canard 양력 분담

  • 주날개와 함께 양력 분담 (trim도 포함)
  • 높은 AR, 캠버 있음
  • 연료 절약 가능 (꼬리 다운로드 없음)
  • 안정성 문제 → 아래 항목 참조

Canard 안정성 문제 ★출제

  • 정적 종안정성(Static Longitudinal Stability): dCm/dCL < 0 필요
  • Canard가 실속하면 → 기수 내려감 (nose down) → 회복 가능
  • 그러나 주날개가 먼저 실속하면 → canard 여전히 작동 → 기수 올라감 (pitch-up) → 위험
  • 해결책: Canard를 항상 주날개보다 먼저 실속하도록 설계 (Canard t/c·AoA 설정)
  • 전통 미익 대비 CG 범위 좁아지는 경향
19 스핀 회복 &
수직꼬리 1/3 규칙

스핀(Spin) 회복에서 수직꼬리(방향타)의 역할이 결정적이다. 스핀 중 수평꼬리 wake가 방향타 일부를 가리기 때문에, 방향타의 최소 1/3은 수평꼬리 wake 바깥에 노출되어야 스핀 회복이 가능하다.

스핀 회복 조건: 방향타 중 ≥ 1/3 이 수평꼬리 후류(wake) 밖에 위치 FAR 규정 기반

T-tail의 스핀 회복 장점

  • 수평꼬리가 수직꼬리 위에 위치
  • 방향타 전체가 wake 밖에 노출
  • 스핀 회복에 유리

T-tail의 Deep Stall 위험

  • 고 AoA: 주날개 후류가 수평꼬리 완전 차단
  • 엘리베이터 효율 상실 → Pitch-up 고착
  • 출구 없는 실속 (deep stall / locked-in stall)

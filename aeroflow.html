01 · CFD 기초 이론 — 지배 방정식

CFD의 모든 것은 유체의 물리 법칙을 수식으로 표현한 지배 방정식에서 출발합니다.

방정식수식 / 형태비고
연속 방정식∇·u = 0 (비압축성)실내 공기, 마하 < 0.3 → 비압축 가정
나비에-스토크스ρ(∂u/∂t + u·∇u) = −∇p + μ∇²u + f점성·압력·부력 포함
에너지 방정식∂T/∂t + u·∇T = α∇²T난방/냉방 동시 해석 시 필요
k-ε 표준 모델난류 운동 에너지 k + 소산율 ε완전 발달 실내 난류에 적합
RNG k-ε 모델표준 k-ε 개선판실내 환기 해석 권장 모델
k-ω SST 모델벽면 + 자유 흐름 동시 정확노즐 출구 복잡한 분리류에 유리
스칼라 수송∂C/∂t + u·∇C = D∇²CCO₂, PM2.5 농도 분포 계산

레이놀즈 수 — 유동 상태 판단

Reynolds Number
Re = ρuL / μ = uL / ν

예시: 노즐 u = 0.5 m/s, L = 0.05 m → Re ≈ 1,700 (층류/난류 전이 영역)
실내 에어컨 출구: Re ≈ 3,000 ~ 20,000 (난류 영역)

k-ε 난류 모델 수식

Turbulence Kinetic Energy k
∂k/∂t + u·∇k = ∇·[(ν + νₜ/σₖ)∇k] + Pₖ − ε

표준 상수: C_μ=0.09, C₁_ε=1.44, C₂_ε=1.92, σₖ=1.0, σ_ε=1.3

02 · OpenFOAM 실습 로드맵

왜 OpenFOAM인가?

무료 오픈소스 CFD. 실제 연구·산업 현장 광범위 사용. HVAC·실내 환기 튜토리얼 풍부. Python으로 자동화 가능. ANSYS Fluent 대비 라이선스 비용 없음.

S0
환경 설치
Ubuntu 22.04 LTS + OpenFOAM v2312 설치. WSL2(Windows) 또는 VirtualBox로 실행 가능. openfoam.com 공식 튜토리얼 가이드 필독.
1주
S1
기본 구조 이해
0/(경계조건), constant/(물성·메쉬), system/(솔버 설정) 폴더 구조 이해. icoFoam으로 단순 채널 유동 첫 실습.
1주
S2
단순 방 모델링
blockMesh로 직육면체 방 생성. 입구(inlet)/출구(outlet)/벽(wall) 경계조건 설정. simpleFoam으로 정상 상태 풀기.
1~2주
S3
난류 + 온도 해석
buoyantBoussinesqSimpleFoam으로 온도 효과 포함. RNG k-ε 모델 적용. ParaView로 속도·압력·온도 등고선 시각화.

S4
오염물 농도 해석
passive scalar로 CO₂ 농도 분포 계산. 공기 연령(Mean Age of Air) 지표 계산 및 데드존 지도 작성. 핵심 발표 자료 도출.
1~2주
S5
가구 장애물 포함
snappyHexMesh로 소파·책상 형상 메쉬 생성. STL 파일 임포트. 장애물 유무에 따른 데드존 면적 비교.
2주
S6
노즐 각도 파라메트릭 최적화
노즐 상하·좌우 각도를 변수로 반복 시뮬레이션. 데드존을 가장 줄이는 최적각 도출 → 논문/발표의 핵심 수치.

03 · 드론 제어 → 기류 제어 응용

핵심 차별점

드론 자세 제어(자세각 유지)와 Aero-Flow AX(실내 기류 방향 유지)는 동일한 수학적 프레임워크를 사용합니다. 상태 공간, 피드백 제어, 최적 제어이론이 그대로 적용됩니다.

상태 공간 표현

항목Aero-Flow AX 정의
상태 변수 x각 구역의 [CO₂ 농도 C₁...Cₙ, PM2.5 D₁...Dₙ, 유속 벡터 u₁...uₙ]
제어 입력 u각 노즐의 [상하 각도 θᵥ, 좌우 각도 θₕ, 팬 RPM n]
출력 yIoT 센서 실측값 (CO₂, PM2.5, 온도, 풍속)
시스템 방정식ẋ = Ax + Bu  (CFD 결과로 선형화한 A, B 행렬)
출력 방정식y = Cx + Du  (센서 배치에 따른 C 행렬)

LQR 최적 제어

LQR 비용 함수 — 최소화
J = ∫₀^∞ [xᵀQx + uᵀRu] dt

Q: 상태 비용 (CO₂ 농도 편차에 큰 가중치 → 엄격히 규제)
R: 제어 비용 (R 클수록 노즐 작동 최소화 → 에너지 절약)

최적 게인: K = R⁻¹BᵀP    최적 제어: u*(t) = −Kx(t)

Python으로 LQR 구현 (scipy)

from scipy.linalg import solve_continuous_are
import numpy as np

# CFD 결과로 선형화한 시스템 행렬
A = np.array([...])
B = np.array([...])  # 노즐·팬 제어 입력

# 비용 행렬 설계
Q = np.diag([10, 10, 1, 1])  # CO₂·PM2.5 편차에 큰 가중치
R = np.diag([1, 1])           # 제어 비용 (소음·전력 절약)

# 연속 Riccati 방정식 풀기
P = solve_continuous_are(A, B, Q, R)
K = np.linalg.inv(R) @ B.T @ P

# 실시간 제어 루프
def control_step(sensor_data):
    x = state_from_sensors(sensor_data)
    u_star = -K @ x  # 최적 제어 입력
    return apply_to_nozzle(u_star)

MPC 예측 제어 — 선제 대응

"요리 시작 전에 미리 기류를 형성"하는 원리

MPC는 N 스텝 앞을 내다보고 최적 제어 계획을 수립합니다. 드론의 경로 계획에서도 동일하게 사용되는 개념입니다.
평소 오후 6시에 요리 패턴이 있다면 → 5시 50분부터 주방 방향 기류를 미리 형성합니다.

04 · 데드존 정량화 방법

지표정의 및 활용
공기 연령 (MAA)외부 공기가 특정 지점에 도달하는 평균 시간. 값이 클수록 데드존. OpenFOAM passive scalar로 직접 계산.
환기 효율 지수 (ε)ε = τₙ/τₗ. 1보다 크면 완전 혼합 이상 효율. 노즐 각도별 효율 비교에 사용.
국소 오염물 제거 효율오염원 위치별 배출 효율 비교. 어느 위치의 오염물이 가장 빨리 배출되는지 지도 작성.
CFD 트레이서 가스법가상 오염물 방출 후 농도 시간 변화 추적. 데드존 면적 히트맵 생성 → 발표 핵심 그림.
공기 연령 수송 방정식
∂(ρτ)/∂t + ∇·(ρuτ) = ∇·(ρD∇τ) + ρ

τ = 공기 연령 [s]. 값이 클수록 오래된 공기 = 데드존.

발표 핵심 포인트 4가지

① CFD 시뮬레이션 영상/이미지로 데드존을 '직접 보여주기'
② 데드존 감소율을 숫자로 제시 (예: 기존 대비 37% 감소)
③ 드론 제어 알고리즘과의 연결 고리를 그림 한 장으로 설명
④ '오늘 당장 사고 싶다'는 사용자 공감 시나리오(페르소나) 제시

02 / AI Model
YOLOv8 커스텀 클래스 (9개)

일반 YOLOv8(COCO 80종)을 직접 데이터 수집·라벨링하여 아래 9개 커스텀 클래스로 파인튜닝. TFLite 변환 후 Raspberry Pi 4B 온디바이스에서 10~15fps 실시간 추론.

06 / AI Layer
AI 연계 단계
1단계 — 현재 구현

실시간 감지 + 즉시 제재

YOLOv8 커스텀 파인튜닝으로 헬멧·인원·노면·신호를 단일 패스 탐지. 위반 즉시 잠금·속도 제어 + 벌점 부여.

2단계 — 확장 방향

위반 패턴 학습 → 예측 제어

사용자별 위반 이력 누적 학습(LSTM). 고위험 사용자 선제 경고, 구역별 히트맵 생성, 보험사 연계 할인 모델.

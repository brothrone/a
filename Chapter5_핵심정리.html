01 T/W · W/S 결정
두 가지 접근법

T/W와 W/S는 항공기 성능에 영향을 주는 가장 중요한 두 파라미터이며, 서로 밀접하게 연결되어 있다. 초기 설계에서 두 값을 동시에 최적화하기 전에 한쪽을 먼저 결정하는 두 가지 방법이 있다.

방법 ① W/S 먼저 결정 실속속도 기준

  • 실속 속도 요구조건으로 W/S 추정
  • 결정된 W/S로 각 성능 조건에서 필요 T/W 계산
  • 이착륙 거리, 상승률 등에서 요구 T/W 도출

방법 ② T/W 먼저 결정 통계값 기준

  • 유사 항공기 통계로 T/W 초기 추정
  • 결정된 T/W로 각 성능 조건에서 W/S 계산
  • 여러 W/S 중 가장 낮은 값 선택 → 날개가 모든 조건에서 충분히 크도록
02 T/W 정의 및
항공기 유형별 통계값

T/W는 해수면, 정지상태, 표준대기, 설계 이륙중량, 최대 추력(에프터버너 포함) 조건을 기준으로 정의된다. 비행 중 T/W가 변하면 반드시 이륙 조건으로 환산해야 한다.

항공기 유형Typical T/W
Jet trainer0.4
Jet fighter (dogfighter)0.9
Jet fighter (other)0.6
Military cargo / bomber0.25
Jet transport (엔진 수↑ → T/W↓)0.25 – 0.4

※ 쌍발 여객기 T/W ≈ 0.4, 4발 T/W ≈ 0.25 (1발 고장 시 0.2로 상승 가능해야 함)

03 프로펠러기
Power Loading

프로펠러기는 엔진 회사가 추력이 아닌 마력(hp)을 판매하므로 Power Loading W/P를 사용한다. 높은 W/P = 작은 엔진을 의미하며 보통 10–15 lb/hp 수준.

T/W = (ηp/V)(P/W) = (550ηp/V)(hp/W) 프로펠러 효율 ηp

※ ηp = 프로펠러 효율 (추력×속도 / 실제 엔진 출력). 속도 증가 시 추력은 감소함

05 Thrust Matching
(5.2.4)

정속 수평 비행에서 추력 = 항력, 양력 = 무게이므로 다음이 성립한다.

(T/W)cruise = 1 / (L/D)cruise Eq. 5.2
(T/W)climb = 1/(L/D)climb + Vvertical/V Eq. 5.3
  1. ①
    L/D 추정 — Wetted aspect ratio → max L/D → 제트기 cruise L/D = 0.866 × max L/D
  2. ②
    T/W 최종 선택 — 통계값 vs 추력매칭값 중 높은 쪽 선택
  3. ③
    이유 — 상승에 충분한 추력이 있으면 동일 고도 순항도 가능; 엔진을 max throttle로 장시간 운용 불가
06 T/W 이륙 조건
환산 (5.2.5)

계산된 모든 T/W는 이륙 조건으로 환산해야 비교 가능하다. 비행 중 연료가 소모되어 무게가 감소하고, 고도·속도에 따라 추력도 변하기 때문.

(T/W)takeoff = (T/W)cruise × (Wcruise/WTO) / (Tcruise/TSLS) 일반 환산식
엔진 유형순항 추력 / 이륙 추력
고바이패스비 터보팬 (여객기)약 20 – 25 %
저바이패스비 애프터버닝 터보팬·터보젯 (전투기)약 40 – 70 %
피스톤 엔진 (비과급)밀도비 σ에 비례하여 감소
터보프롭 (등가 축마력, eshp)순항 eshp ≈ 이륙의 60 – 80 %

※ 이륙→상승 무게비: 0.97×0.985 = 0.956 → 순항 T/W에 0.956 곱하면 이륙 기준 무게 반영

07 Wing Loading W/S
개요 및 통계값

W/S = 항공기 중량 / 기준 날개 면적으로, 실속 속도·상승률·이착륙 거리·선회 성능에 직접 영향. 여러 조건에서 구한 W/S 중 가장 낮은 값 선택해야 날개가 모든 비행 조건에서 충분히 크다.

항공기 유형W/S [lb/ft²][kg/m²]
Sailplane6{30}
Homebuilt11{54}
GA — single engine17{83}
GA — twin engine26{127}
Twin turboprop40{195}
Jet trainer50{244}
Jet fighter70{342}
Jet transport / bomber120{586}
08 Stall Speed →
W/S (5.3.2)
W/S = ½ρV²stall · CLmax 실속 시 L = W
  1. ①
    접근 속도 규정 — 민간: Vapproach ≥ 1.3 Vstall / 군용: ≥ 1.2 Vstall
  2. ②
    FAR 23 규정 — TOGW ≤ 12,500 lb 민간기: Vstall ≤ 61 kt (113 km/h)
  3. ③
    후퇴익 CLmax — CLmax = 0.9 · Clmax · cosΛ0.25c
날개 형태CLmax 범위
플랩 없는 순수 날개1.2 – 1.5
일반기 (내측 플랩)1.6 – 2.0
여객기 (플랩 + 슬랫)≈ 2.4
STOL (대형 플랩·슬랫)≈ 3.0
프롭워시·제트워시 침지최대 5.0

※ 이륙 CLmax ≈ 착륙 CLmax × 0.80 (이착륙 설정 다름)

09 이륙 거리 →
W/S (5.3.3)

이륙 거리 개념 3가지: Ground roll (바퀴가 뜨는 순간까지 활주거리), Obstacle clearance distance (군 50ft, 민 35ft 고도 장애물 통과까지), BFL (Balanced Field Length) (엔진 고장 시 계속 이륙 vs 정지 거리가 같아지는 최소 활주로 길이).

TOP (Jet) = (W/S) / (σ · CLTO · T/W) 이륙 파라미터
(W/S) = TOP · σ · CLTO · (T/W) W/S 역산
  1. ①
    Vliftoff = 1.1 × Vstall — 이륙 활주 중 양력 계수: CLTO = CLmax/1.21
  2. ②
    Decision Speed (V₁) — BFL: 정지 거리 = 잔여 엔진 이륙 거리가 되는 속도
  3. ③
    쌍발기 BFL — 3·4발기보다 큼 → 더 높은 총 T/W 필요
10 착륙 거리 →
W/S (5.3.5)
Slanding [ft] ≈ 0.3 · V²approach [knots] 초기 추정
Slanding = 80 · (W/S) · (1/σCLmax) + Sa [ft] Eq. 5.11
  1. Sa
    장애물 통과 거리 Sa — 여객기(3° 진입): 1000 ft / GA 동력 차단: 600 ft / STOL(7°): 450 ft
  2. ①
    FAR 25 (수송기) — 총 착륙 거리 × 1.67 (안전 여유)
  3. ②
    역추력/역피치 — 지상 활주 부분 × 0.66
  4. ③
    이륙 W/S 환산 — (W/S)TO = (W/S)landing × WTO/Wlanding. 제트기: 착륙중량 ≈ 이륙중량의 85%
11 순항 최적 W/S
(5.3.7)

L/D 최대점에서 최적 W/S가 결정된다. 필요 공력 계수: CD0 (영양력 항력계수)와 e (Oswald 효율).

프로펠러기 최대 항속 (L/D max 비행):
W/S = q√(πAeCD0) Eq. 5.13
제트기 최대 항속 (기생항력 = 3×유도항력):
W/S = q√(πAeCD0/3) Eq. 5.14
  1. CD0
    영양력 항력계수 — 제트기 ≈ 0.015 / 청결 프롭기 ≈ 0.02 / 고정 기어 프롭기 ≈ 0.03
  2. e
    Oswald 효율 — 전투기 ≈ 0.6–0.8 / 일반기 ≈ 0.8
  3. q
    동압 q = ½ρV² — 해당 순항 고도·속도 조건
12 체공 최적 W/S
(5.3.8)

체공(Loiter)은 L/D 최대점(제트) 또는 필요 동력 최소점(프롭)에서 최적화된다.

제트기 최대 체공 (L/D max):
W/S = q√(πAeCD0) Eq. 5.15
프로펠러기 최대 체공 (유도항력 = 3×기생항력):
W/S = q√(3πAeCD0) Eq. 5.16
(W/S)takeoff = (W/S)loiter × WTO/Wloiter 이륙 조건 환산 (비율 ≈ 0.85)

※ 체공 최적 고도: 제트 30,000–40,000 ft, 프롭 과급기 한계 고도 이내

13 순간 선회
(5.3.9)

순간 선회율(Instantaneous Turn Rate): 속도·고도 유지를 무시하고 최대 선회율만 추구. 날개 실속 또는 구조 하중 한계에 의해 제한됨.

ψ̇ = g√(n²−1) / V [rad/s] Eq. 5.17
n = qCL / (W/S) (하중 계수 = Load Factor) Eq. 5.18
V* = √(2nmax/(ρ·CLmax) · W/S) (코너 속도, 최대 선회율 속도) Corner Speed
W/S = qCLmax / n Eq. 5.20
  1. n
    최대 하중 계수 — 구형 전투기: 7.33g / 신형 전투기: 8g
  2. Wc
    전투 중량 — 외부 탱크 투하 + 내부 연료 50% 소모 ≈ 이륙중량의 0.85배
  3. CL
    전투 CLmax — 단순 후연 플랩: 0.6–0.8 / 전·후연 복합: 1.0–1.5
14 지속 선회
(5.3.10)

지속 선회(Sustained Turn): 속도·고도를 유지하며 선회 (추력 = 항력). L/D 최대점에서 하중 계수 최대.

n = (T/W)(L/D) Eq. 5.21
W/S = (q/n)√(πAeCD0) Eq. 5.22 (n=1이면 순항 최적식과 동일)
T/W = qCD0/(W/S) + (W/S)(n²/qπAe) Eq. 5.24 W/S 역산 가능
W/S = [(T/W) ± √((T/W)² − 4n²CD0/πAe)] / (2n²/qπAe) Eq. 5.25 요구 지속 선회 n 달성 W/S

※ 순간 선회: 짧은 시간 최대 선회율 / 지속 선회: 속도·고도 유지하며 장시간 선회

15 W/S 최종 결정
절차 요약
  1. 1
    실속 속도 기준 W/S — FAR 규정 또는 접근속도 요구로 산출
  2. 2
    이륙 거리 기준 W/S — TOP 그래프에서 TOP 읽고 W/S = TOP×σ×CLTO×T/W
  3. 3
    착륙 거리 기준 W/S — Eq. 5.11 역산 후 이륙 조건으로 환산 (÷ Wland/WTO)
  4. 4
    순항 최적 W/S — Eq. 5.13 or 5.14 (제트/프롭 구분)
  5. 5
    선회 기준 W/S — 순간/지속 선회 요구조건에서 산출 후 이륙 조건 환산
  6. ★
    최종 선택: 가장 낮은 W/S — 모든 조건에서 날개가 충분히 크도록 보장

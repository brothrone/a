01 항공기 설계 및
개발 과정 5가지
  1. 1
    Requirements — 새로운 개념·아이디어 입력
  2. 2
    Conceptual Design — 요구사항 기반 형상 결정, 성능·비용 검토
  3. 3
    Preliminary Design — 형상 확정(freeze), lofting, 비용 추정
  4. 4
    Detail Design — 실제 부품 설계, 구조·착륙장치 시험, 중량·성능 확정
  5. 5
    Fabrication & Ground/Flight Test — 제작 및 지상·비행 시험
02 설계중량 결정
요소 4가지
  1. 1
    Payload weight — 탑재 화물·승객 중량
  2. 2
    Fuel weight — 임무 수행 필요 연료 중량
  3. 3
    Empty weight — 구조·엔진·시스템 자체 중량
  4. 4
    Crew weight — 승무원 중량
03 Reference Wing
결정 요소 3가지
  1. 1
    Wing Area (S) — 기준 날개 면적
  2. 2
    Aspect Ratio (AR) — 가로세로비, b²/S
  3. 3
    Taper Ratio (λ) — 테이퍼비, tip chord / root chord
04 Dihedral 효과를
주는 요소 3가지
  1. 1
    Dihedral angle — 날개 자체의 상반각
  2. 2
    Wing sweep (후퇴각) — 롤 시 상반각 효과 발생
  3. 3
    High wing configuration — 동체 위 날개 위치 → 펜들럼 안정성
05 MAC와 Pitching
Moment 관계

Aerodynamic Center(AC)는 받음각(AoA)이 변해도 pitching moment가 변하지 않는 점이다.
MAC는 테이퍼·후퇴각을 고려한 3D 날개의 등가 평균 시위선으로, 그 25% 위치가 전체 날개의 AC로 사용된다.
CG가 AC 앞 → 안정 (nose-down moment) / CG가 AC 뒤 → 불안정 (발산)

06 고익기·저익기
이유와 설계상 장점

고익기 (High Wing) 군용 수송기

  • 동체를 지면에 가깝게 → 화물 적재·하역 용이
  • 랜딩기어 짧게 → 중량 감소
  • 날개 박스가 동체 위 통과 → 내부 공간 확보
  • 자연적 상반각 효과 → 횡방향 안정성↑

저익기 (Low Wing) 민간 여객기

  • 객실 공간 방해 없음 (wing box 동체 아래)
  • 착륙장치를 날개에 수납 → 구조 효율적
  • 날개 연료 → 굽힘 모멘트 경감 (bending relief)
  • 대형 수송기: 객실 위 wing carrythrough box
07 Wing Loading
결정 요소 3가지
  1. 1
    Cruise performance — 순항 속도·고도에서 L/D 최적화
  2. 2
    Stall / T-O / Landing — 낮은 W/S → 저속 이착륙 가능
  3. 3
    Maneuverability — 전투기는 높은 W/S로 고속 기동
08 꼬리날개 역할
3가지
  1. 1
    종방향 안정·조종 — 수평 꼬리날개 → 피칭(Pitching) 제어
  2. 2
    방향 안정·조종 — 수직 꼬리날개 → 요잉(Yawing) 제어
  3. 3
    트림 (Trim) — 다양한 비행 조건에서 모멘트 균형 유지
